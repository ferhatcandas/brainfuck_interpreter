# brainfuck_interpreter


# Installation

```code 
go get github.com/ferhatcandas/brainfuck_interpreter
```

# Execute from cli 

```code 
./brainfuck_interpreter exec "++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+."
```

# Execute from file 

```code
./brainfuck_interpreter file "./sample.bf"
```
